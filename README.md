# Predicting CO2 Emission of Vehicles Using Machine Learning

## Overview
This project predicts vehicle CO2 emissions using machine learning regression models.
It demonstrates data preprocessing, model training, and prediction using Python.

## Technologies Used
- Python
- Scikit-learn
- Pandas
- Random Forest Regressor
- Streamlit

## Features
- CO2 emission prediction based on vehicle parameters
- Machine learning regression model
- Simple interactive web interface

## How to Run
1. Install dependencies:
   pip install -r requirements.txt
2. Train the model:
   python train_model.py
3. Run the web app:
   streamlit run app.py
